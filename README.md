## Comprehension Check
You will be employer-ready if you can answer the following questions:

1. What does it mean to be a full-stack web developer?
- A full stack web developer is a person who can develop front and back ends of a website or application.
2. What is the relationship between HTML and CSS?
- HTML is used to create content of the page, such as written text.
- CSS is used to create a design or style of the website.
3. What is Git workflow?
-  Git offers a lot of flexibility in how developers can manage changes on their code.
